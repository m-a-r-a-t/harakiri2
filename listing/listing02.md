Что выведет программа? Объяснить вывод программы. Объяснить как работают defer’ы и их порядок вызовов.

```go
package main

import (
	"fmt"
)


func test() (x int) {
	defer func() {
		x++
	}()
	x = 1
	return
}


func anotherTest() int {
	var x int
	defer func() {
		x++
	}()
	x = 1
	return x
}


func main() {
	fmt.Println(test())
	fmt.Println(anotherTest())
}
```

Ответ:
```
В первом случае будет 2 так как в итоге мы возвращаем значение именованной переменной, соотвественной после return,
оператор defer меняет значение переменной x.

Во втором случае будет 1 так как возвращаем значение и оператор defer уже никак не влияет на изменение этого результата.

defer - это оператор для отложенного запуска функции. Функция отложенная через этот оператор выполнится после завершения функции ,где был
сделан отложенный запуск. Если отложено много функций,то те которые были объявлены позже выполнятся первыми. 

```
